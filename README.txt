# Klasifikasi Rempah Indonesia 🇮🇩

Proyek ini adalah aplikasi berbasis Streamlit untuk mengklasifikasikan jenis rempah-rempah Indonesia menggunakan EfficientNetV2-S.

## 🔍 Fitur
- Upload gambar rempah (bubuk atau utuh)
- Prediksi jenis rempah secara instan
- Model deep learning di-hosting via Google Drive
